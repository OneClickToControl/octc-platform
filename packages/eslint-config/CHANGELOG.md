# @1c2c/eslint-config

## 0.2.0

### Minor Changes

- 57aa0c4: Initial public release of `@1c2c/tsconfig` and `@1c2c/eslint-config`.

  - `@1c2c/tsconfig`: `base`, `library`, `nextjs` configurations targeting modern Node, library and Next.js stacks.
  - `@1c2c/eslint-config`: flat-config presets for general TypeScript projects, libraries and Next.js apps.

  Both packages are published to npm with SLSA v1 provenance via GitHub Actions OIDC. See `docs/security/SUPPLY_CHAIN.md`.
